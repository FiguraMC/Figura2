# This is not the main Figura Repository!
The current Figura repository is located at [https://github.com/FiguraMC/Figura](). 
Code in this repository is intended to eventually become part of a later version of 
Figura, but it is currently heavily WIP.

## Why is this public?
This repository was made public, despite its WIP nature, to allow Figura's community
members to view the code and provide their thoughts.
